Objective : Predicting Home prices in SUburbs of Boston

The 'Project_Home_prices.RMD' file illustrates the step by step procedure to fit an appropriate model on the Boston dataset to predict the median home prices in different suburbs of Boston.The RMD file is self sufficient and can be run to fit models and produce results.

Step 1 :  Run the script "install_list_of_packages.R" to install all the required packages and their dependencies.

Step 2 : Knit 'Project_Home_prices.RMD'

Step 3 : Running Step 2 results in 'Project_Home_prices.html' being created.
